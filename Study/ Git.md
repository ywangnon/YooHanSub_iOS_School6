#Git
##What is Git?
- 소스 코드 관리를 위한 분산 버전 관리 시스템
- 변경 사항을 추적하여 코드의 히스토리 관리

##Git의 구조
###1. Work Space
- 실제 작업 공간

###2. index
- Work Space와 내부 저장소의 중간 단계 공간
- 일종의 준비 구역
- staging area

###3. 내부저장소
- git이 설치된 컴퓨터 폴더
- Local Repository

###4. 외부저장소
- 서버에 존재
- 외부에 저장하는 곳
 
###*Github는 Git 서비스 중 하나일 뿐이다.
